## DVLD Project
**Driver and Vehicle Licensing Department System**

---

### 📝 Overview
A specialized management system designed to streamline the processes of driver licensing and vehicle registration, ensuring efficient data handling and a smooth user workflow.

### 🛠 Technical Stack
* **Language:** C#
* **Framework:** .NET (WinForms)
* **UI Library:** SunnyUI (Modernized components and controls)
* **Database:** SQL Server
* **Data Access:** ADO.NET
